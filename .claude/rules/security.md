# 安全规则（必须遵循）

## 强制检查（提交前）
- [ ] 禁止硬编码密钥（API Key/Token/密码）
- [ ] 所有用户输入必须校验
- [ ] SQL 注入防护（参数化查询）
- [ ] XSS 防护（输出转义/清理）
- [ ] CSRF 保护（Web 表单/状态变更）
- [ ] 鉴权与授权必须完整
- [ ] 关键端点启用速率限制
- [ ] 错误信息不泄露敏感数据

## 密钥管理
- 使用环境变量或密钥管理服务
- 运行时验证必需环境变量
- 生产密钥与测试密钥严格隔离

## 安全响应
发现安全问题：立即停止 → 使用 security-reviewer → 修复后继续
